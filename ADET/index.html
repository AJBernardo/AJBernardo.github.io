ADET PROJECTS

Welcome to my collection of applications development and emerging technology projects! These projects showcase my expertise in building innovative applications while integrating the latest advancements in technology. From AI-powered solutions to modern software development, explore how I leverage cutting-edge tools to create impactful digital experiences.

01 CAFE LYCORECO USING MVC MODEL

Café LycoReco is a character-themed web project based on the anime Lycoris Recoil, built using the MVC architecture to separate content, layout, and logic. It features dynamic pages for Home, Menu, Staff, and FAQs, capturing the warm, welcoming spirit of the café through clean and organized development.

02 DEDSEC: BENTO LAYOUT

DedSec Rebellion Hub is a web project inspired by the Watch Dogs universe, designed to echo DedSec’s spirit of digital freedom. Built with a clean structure and enhanced by a Bento-style vision board, the site delivers bold visuals and a strong message of resistance, blending static and dynamic elements into one unified experience.

03 NAMI: POINT OF SALE SYSTEM

NAMI is a sushi-focused Point of Sale system designed for simplicity and speed. Developed primarily with HTML, CSS, and JavaScript, it features a clean, responsive interface with dynamic category filtering and real-time item tracking. NAMI brings efficiency to the sushi-serving experience while maintaining a smooth and intuitive workflow.

04 IDOLFACTORY WITH EMBEDDABLE CALENDAR

This project highlights the talented actors of IdolFactory Thailand, featuring detailed profiles, career highlights, and their most popular dramas. It also includes an embeddable calendar that allows fans to track upcoming series, actor appearances, and special events—making it easier to stay updated with everything IdolFactory in one interactive hub.
